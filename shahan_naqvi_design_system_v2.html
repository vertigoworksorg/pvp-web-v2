## ShananNaqvi Personal Brand Design System v2

### Identity
Name: Shahan Naqvi  |  Role: Software Engineer · AI Builder · Building in Public
Niche: AI-powered productivity, self-discipline, open-source development
Audience: Developers, tech professionals, productivity enthusiasts
Voice: Bold, direct, relatable, authentic builder

### Color Palette
Primary Blue:    #1B5AFF  (Electric Blue — CTAs, names, key headlines)
Blue Light:      #EEF3FF / #D6E4FF / #ADCBFF  (fills, borders, hover)

Accent Indigo:   #6457E8  (secondary accent — tags, highlights, progress)
Indigo Light:    #EEF0FF / #DDD9FF / #C0BAFF  (fills, pill tags)

Background:      #FAFBFD (page), #FFFFFF (cards), #F0F2F7 (subtle areas)
Dividers:        #E2E6EF (borders), #CBD2E0 (stronger borders)

Text Heading:    #0D1136 (primary), #1A1F3C (secondary heading)
Text Body:       #4A5174 (body), #6B7594 (muted), #9AA3BC (subtle/caption)

Semantic:        #10B981 success, #F59E0B warning, #EF4444 error

### Typography
Display / H1-H3: Fraunces (600 display, 500 h2, 400 h3) — elegant, refined
Body / UI / Tags: DM Sans (400 body, 500 label, 600 badge)
Both from Google Fonts — import both in every design.

### Signature Design Elements
- White + soft gray backgrounds ONLY (no dark mode, no navy BG)
- Subtle dot grid overlay (#1B5AFF, 10-12% opacity) on hero surfaces
- Vertical 3.5px blue (#1B5AFF) left-side bar beside name/headline
- Indigo dot/circle accent at top of the vertical bar
- Soft radial gradient blobs (blue + indigo, very low opacity: 8-12%)
- Thin horizontal hairlines (#1B5AFF 12% + #6457E8 12%) across sections
- Crosshair (+) accent marks in corners — subtle, not distracting
- Floating small squares/dots in blue and indigo (opacity 15-20%)
- Blue tags (#EEF3FF bg, #ADCBFF border, #0A3CC7 text)
- Indigo tags (#EEF0FF bg, #C0BAFF border, #4C3DD4 text)
- Progress bar: #E2E6EF track, #1B5AFF fill, #6457E8 dot at end
- Translucent white cards (opacity 0.75-0.85) on gradient surfaces
- Dashed divider lines between left and right content zones

### Content Pillars
1. 100 Days of Rebuilding Focus (active project — always feature)
2. AI Self-Discipline Coaching Tool (building in public)
3. Software engineering / full-stack insights
4. Productivity, deep work, atomic habits
5. Open source culture

### Asset Sizes
LinkedIn Banner:  1584 × 396px  |  LinkedIn Post: 1080 × 1080px
Twitter/X Header: 1500 × 500px  |  Website Hero:  1440 × 800px
Story / Reel:     1080 × 1920px |  Article Cover: 1280 × 720px

### Design Rules (Never Break)
✦ Background is ALWAYS white or soft gray (#F0F2F7 / #FAFBFD)
✦ Blue (#1B5AFF) = primary actions, names, key accents
✦ Indigo (#6457E8) = secondary accent, tags, feature highlights
✦ Blue tags for tech/role items · Indigo tags for content/topic items
✦ Fraunces for any display headline · DM Sans for everything else
✦ Pill border-radius on ALL buttons and tags (border-radius: 999px)
✦ Always include "Currently Building" section in banners
✦ Cards use white bg + subtle border (not colored backgrounds)
✦ No cyan, no dark backgrounds, no neon glow effects
✦ Gradient text is allowed only for display headlines (blue to indigo)
